# sample-online-jsf
JavaEE7 JSFを利用したサンプルWebアプリケーションです。

## 動作確認環境
| Platform | Version | Edition |
| -------- | ------- | ------- |
| IBM WebSphere Application Server | 18.0.0.3 | Liberty |
| IBM Java | 1.8.0_181 | |

| FW/OSS | Version |
| ------ | ------- |
| JavaEE | 7 |
| JSF (Apache MyFaces) | 2.2 |
| logback | 1.2.3 |
| jQuery | 3.4.1 |
